DwebBrowser 前后端打包工具，把您开发的包打包成`.bfsa`形式。

### @bfex/bundle

```
 全局安装指令
 # npm方式
   npm i @bfex/bundle -g

 # yarn方式
   yarn global add @bfex/bundle
```

#### 使用

安装完直接运行bfex，开启交互式打包

```bfex
bfex
```
